# 🧪 EMoov - Quality Assurance Case Study

Este repositório reúne a documentação de **Quality Assurance (QA)** elaborada para o projeto **EMoov**, um sistema de mobilidade interna e controle de frota desenvolvido como Trabalho de Conclusão de Curso (TCC).

O objetivo deste projeto é demonstrar o planejamento, a execução e a documentação de testes manuais realizados sobre uma aplicação real, utilizando boas práticas de QA para validar funcionalidades críticas do sistema.

---

## 📌 Sobre o EMoov

O **EMoov** é uma plataforma desenvolvida para otimizar a mobilidade interna e o gerenciamento da frota de veículos da Prefeitura Municipal de Santana de Parnaíba.

A solução é composta por:

- 🖥️ Painel Administrativo Web
- 👤 Aplicativo do Funcionário
- 🚗 Aplicativo do Motorista
- ⚙️ Backend REST API

Entre suas principais funcionalidades estão:

- Gestão de veículos
- Gestão de motoristas
- Gestão de funcionários
- Solicitação de corridas
- Agendamento de corridas
- Controle da frota
- Comunicação em tempo real entre motorista e solicitante
